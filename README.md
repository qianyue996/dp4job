### How to use

#### 制作向量数据库

运行`make_database.py`

#### 前后端分别运行
运行`vllm_server_api.py`启动后端，监听`127.0.0.1:8000`

运行`vllm_webui.py`启动前端，监听`0.0.0.0:7860`
